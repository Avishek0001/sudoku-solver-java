# sudoku-solver-java -
A sudoku can be solved by using backtracking in recursion. The Code has been done in Java Language. There are some rules without which we cannot solve a sudoku and by using this code any sudoku can be solved.
-> There will be a 9x9 grid which will contain 9 3x3 grids.
-> Only use the numbers 1 to 9 in a game of Sudoku.
-> Only use each number once in each row, column, & grid
